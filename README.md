# Sales Data Analysis using Python

## Project Overview

This project analyzes a retail sales dataset using Python to identify sales trends, monthly performance, customer group performance, and state-wise sales. The project includes data cleaning, normalization, descriptive statistics, and data visualization to generate meaningful business insights.

---

## Objectives

- Clean and prepare the dataset for analysis.
- Normalize numerical features using Min-Max Scaling.
- Analyze daily sales and unit trends.
- Compare monthly sales performance.
- Study sales distribution using histograms and box plots.
- Perform state-wise and customer group-wise analysis.
- Analyze sales trends over time.
- Summarize key business insights using visualizations.

## Project Workflow

### 1. Data Preparation
- Checked column names
- Converted the Date column to datetime format
- Verified dataset information

### 2. Data Normalization
- Applied Min-Max Scaling to Unit and Sales columns

### 3. Daily Sales Analysis
- Grouped data by Date
- Visualized Daily Sales and Daily Units Sold using line charts

### 4. Monthly Analysis
- Split the dataset into October, November, and December
- Calculated descriptive statistics
- Compared monthly sales performance

### 5. Distribution Analysis
- Histogram
- Box Plot
- Identified sales distribution and outliers for each month

### 6. State-wise Analysis
- Calculated total Units and Sales by State
- Compared states using bar charts

### 7. Group-wise Analysis
- Calculated total Units and Sales by customer group
- Compared customer groups using bar charts

### 8. Time-wise Analysis
- Analyzed sales by day of the week
- Visualized trends using line charts

---

## Key Findings

- Sales remained relatively stable during October.
- November recorded the lowest average sales.
- December achieved the highest sales among all three months.
- November contained two sales outliers.
- December contained one unusually low sales outlier.
- State-wise and group-wise analysis identified the highest and lowest performing categories.
- Daily sales trends showed a decline during November followed by a strong increase in December.
